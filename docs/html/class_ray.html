Detailed Description

template<typename T, typename U>
class Ray< T, U >

Represents a ray in an optical system.

The Ray class encapsulates the properties of a ray of light, including its position, direction, wavelength, and intensity. The direction vector is maintained as a normalized vector to ensure accurate representation of the ray's path.

Template Parameters:

  • T: Type for position and direction vector components (e.g., float, double).
  • U: Type for wavelength and intensity (e.g., int, float).
